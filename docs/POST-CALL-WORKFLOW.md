## üìã Step 2: Create Custom Proposal

**Send within 24 hours.**

### What to Include:

1. **Executive Summary** (2-3 sentences)
   - Their problem
   - Your solution
   - Expected outcome

2. **Scope of Work** (Bullet points)
   - Exactly what's included
   - What's NOT included (set boundaries)
   - Timeline with milestones

3. **Investment** (Clear pricing)
   - Package name
   - Total price
   - Payment options (full/deposit/installment)
   - What they get for the money

4. **Your Process** (Build trust)
   - Phase 1: Discovery & Planning
   - Phase 2: Design
   - Phase 3: Development
   - Phase 4: Launch & Training

5. **Next Steps** (Clear CTA)
   - **BIG BUTTON: "Accept Proposal & Pay"**
   - Secure payment link (Stripe)
   - Easy to click on mobile

6. **FAQ** (Answer objections)
   - "What if I need changes?"
   - "When do I pay?"
   - "How long does it take?"

7. **Social Proof** (Build confidence)
   - 1-2 testimonials
   - Example websites you've built
   - Your guarantee

## üîó Step 3: Generate Stripe Payment Link

### Using the API:

```javascript
// In your admin dashboard or via API
const response = await fetch('/api/create-payment-link', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        customerName: 'John Doe',
        customerEmail: 'john@example.com',
        amount: 5000, // $5,000
        description: 'Professional Website Package - John\'s Restaurant',
        metadata: {
            callDate: '2026-02-16',
            package: 'Professional',
            features: 'E-commerce, Blog, Booking System'
        }
    })
});

const { paymentUrl } = await response.json();
// paymentUrl = "https://buy.stripe.com/xxxxx"
```

### Or Use Stripe Dashboard:

1. Go to [Stripe Dashboard ‚Üí Payment Links](https://dashboard.stripe.com/payment-links)
2. Click "+ New"
3. Fill in:
   - **Product name**: "Professional Website - [Business Name]"
   - **Price**: $5,000
   - **Description**: Custom website package
4. Click "Create link"
5. Copy the URL
6. Paste into proposal email

## üí° Pro Tips

### Increase Close Rate:

1. **Limited Time Offer** - "This price is valid for 7 days only"
2. **Show Scarcity** - "I only take on 2 projects per month"
3. **Include Bonus** - "Book this week and get a free logo refresh ($500 value)"
4. **Make It Easy** - One-click payment link (not a quote request form)
5. **Remove Risk** - "100% money-back guarantee if not satisfied"

### Track Everything:

- When proposal was sent
- When they opened it (use email tracking)
- When payment link was clicked
- Follow-up dates
- Reasons for lost deals

### Common Objections & Responses:

**"It's too expensive"**
‚Üí "I understand. Let me show you the ROI: If this website brings in just 2 extra customers per month at $X each, it pays for itself in Y months."

**"I need to think about it"**
‚Üí "Of course! What specific concerns do you want to think through? I'm happy to address them now."

**"Can you do it cheaper?"**
‚Üí "I can create a scaled-down version for $X. Here's what would be included: [List]. Would that work better?"

**"I need to talk to my partner"**
‚Üí "Great! Would it help if I joined a call with both of you to answer any questions?"
