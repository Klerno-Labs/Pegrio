## Image Optimization Tips

Before using images:
1. **Resize** to recommended dimensions
2. **Compress** using:
   - TinyPNG.com
   - Squoosh.app
   - ImageOptim (Mac)
3. **Convert to WebP** for best performance
4. **Target size**: < 200KB per image

## Quick Download Instructions
